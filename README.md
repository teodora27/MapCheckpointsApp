## How to Start This Awesome Application 🚀

1. **Step 1**:  
   Head over to the **backend** folder, open a command prompt, and type:  
   `yarn start` 💻

2. **Step 2**:  
   Now, jump into the **frontend** folder, open a command prompt, and run:  
   `yarn start` ⚡

3. **Step 3**:  
   🎉 **Make an account and enjoy!** 🍕 🎮 🍹
